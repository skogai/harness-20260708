# /skogharness:example

Use this as the smallest valid example of a Skogharness command definition.

## Trigger Pattern

```text
/skogharness:example [short task]
```

## Behavior

1. Restate the requested task in one sentence.
2. Identify the files or docs that should be checked first.
3. Recommend the smallest next action.

## Boundaries

- This command is example documentation only.
- It does not install itself or create runtime behavior yet.
